# Sistema Logístico -- Gestión de Inventario con AVL y Colas FIFO

Este proyecto implementa un sistema logístico para administrar lotes de
productos mediante un árbol AVL complementado con colas FIFO de pedidos.

## Características

-   Árbol AVL para administración de lotes por fecha
-   Cola FIFO de pedidos por lote
-   Cancelación de pedidos y eliminación de nodos
-   Reporte in-order del inventario

## Compilación y ejecución

    gcc proyectoFinal.c -o sistema
    ./sistema
